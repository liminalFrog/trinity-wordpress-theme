# Screenshot Information

To add a screenshot for your theme:

1. Create an image that's 1200px × 900px
2. Name it "screenshot.png"
3. Place it in your theme's root directory: /wp-content/themes/mytheme/screenshot.png

This image will appear in the WordPress admin under Appearance > Themes.

For now, you can use any image editing software to create a simple preview of your theme design.
